Weather Station

Weather Station

This is my little Arduino weather station which includes Adafruit’s rgb LCD, a DHT 11 weather/humidity sensor, Arduino, potentiometer and 2 leds indicating if the temperature is below freezing. This program works by first reading a value from the weather sensor and then using calculations to get the value from digital to Celsius temperature and then into Fahrenheit. Then using the LCD library the temperature data is sent to the LCD where it is displayed. This LCD’s backlight can be controlled by the potentiometer on the right. Finally the code checks the temperature in order to determine if you have to wear a jacket and will light the RED LED if you do not and the Green if you do have to wear a jacket.
