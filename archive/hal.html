Spooky Project

Halloween Project

The above project was setup to determine when trick or treaters passs by my door way. When trick or treaters would pass by spooky sounds would play. This system was built first with a set of ultrasonic sensors to determine when the trick or treaters passed by the doorway. These values were sampled via an Arduino digital pin from 3 of the ultrasonic sensors placed on the doorway. After this was complete if a change in value occured in 2 of the 3 sensors then it can be determined that a trick of treater is present. Thus this data was sent over Serial to a RPI where the RPI then would play one of the custom spooky tracks for a select time. Now the tracks played were customly contrallable since the RPI song was controlled via a webpage. This webpage stored the value of the song number to play in a SQL database where the RPI would query this number each time to determine which song to play. Overall this was a really fun project that I was able to build all in one night right before halloween!
