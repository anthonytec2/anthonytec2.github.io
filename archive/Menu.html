Arduino Suite

Arduino Suite

This project was combining two sensors, an ultrasonic sensor and temperature sensor, in a LCD controllable interface. The LCD basically displayed the values of each of these sensors which was controllable via a potentiometer to control which menu item to forward. Additionally, there was a push button to confirm which item to press. Aside from this, based on the sensor values from each sensor this would control the color of the RGB backlight on the LCD.
