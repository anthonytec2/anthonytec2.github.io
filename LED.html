LED Game

The LED game is a game in which you hit a button when an LED is lit and then you advance to the next level. Though I put a new twist on it with the serial in which instead of using a button you have to press the number 1 and enter. At the end of all levels the LED’s go in a sequence. I used an Arduino and 5 LEDs in this version of the game.
