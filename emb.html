Embedded Design

Robotic Arm Final Project

The above is my final project for my embedded design course which introduced principles in digital and software design. The scope of the project involved creating a moving robotic arm with two modes: computer controlled move mode and wii mote controlled mode. The video above demonstrates the computer controlled moving mode which I was able to make the robotic arm throw the box. The design for this system was on a Zedboard which is a FPGA with an additional ARM core for any programming tasks. The FPGA digital design, used in both modes, was developed in Simulink HDL coder which performed clock division and PWM generation. On the other side of things, the ARM core performed state flow to move the arm from position to position.
