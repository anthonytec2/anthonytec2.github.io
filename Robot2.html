Robot Arm Sci-Oly

Robot Arm Sci-Oly

The robot arm that we built for states uses a Pololu Maestro Servo controller, a bunch of servos, female wire connectors, male wire connectors, a passive electromagnet, a custom designed claw, lazy susan joint, computer, aluminum, a USB to mini USB A cable, various screws, velcro and a PS3 controller. The basics behind how the robot arm moves is there is a rotational servo which turns an upper wood disc along a lazy susan joint. In order to make the robot arm move up and down 2 high powered servos are used which are both use the same signal wire. Then to change the angle at which an object is picked up the top servo on the aluminum is used. To close and open the claw a servo is hooked up with wire to open and close the claw based on tension. The same is true for the passive electromagnet. As for the principals of operation for the Robot Arm the PS3 controller is the starting point. The PS3 controller is connected to a computer via bluetooth and the values of all the current analog sticks and buttons pressed are constantly updated. Now based upon those values Serial commands are sent to the servo controller moving the servos.
